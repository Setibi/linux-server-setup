# Linux Server Setup

Configuración básica y segura de un servidor Ubuntu.

## Tecnologías
- Ubuntu Server 24.03 LTS
- OpenSSH
- UFW
- Bash scripting

## Pasos realizados
1. Instalación de Ubuntu y configuración inicial
2. Creación de usuarios y grupos
3. Hardening SSH (deshabilitar root, cambiar puerto)
4. Configuración del firewall (UFW)
5. Scripts de backup
